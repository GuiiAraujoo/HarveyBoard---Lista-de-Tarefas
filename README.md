# 🦊 HarveyBoard – Lista de Tarefas Interativa

O HarveyBoard é mais do que uma simples lista de tarefas.  
Ele é um quadro divertido e interativo, com o Harvey (seu assistente virtual) te acompanhando em cada passo.  
A ideia aqui não foi criar um sistema corporativo cheio de formalidades, mas sim algo **descontraído**, que dá vontade de usar no dia a dia.

---

## 🎯 Minha visão
Quando pensei nesse projeto, eu queria fugir do básico.  
Não queria só uma lista de "adicionar tarefa" e pronto.  
Queria algo que tivesse personalidade que fosse leve e até um pouco engraçado.  
Por isso o Harvey está sempre lá, te dando bom dia, boa tarde ou boa noite, e te guiando de forma amigável.

---

## 🚀 Funcionalidades
- Saudação personalizada de acordo com o horário do dia.
- Coleta do nome do usuário (porque o Harvey gosta de chamar você pelo nome).
- Criação de novas tarefas.
- Edição de tarefas existentes.
- Marcar tarefas como concluídas.
- Excluir tarefas (com envio para a lixeira).
- Restaurar tarefas da lixeira ou das concluídas.
- Apagar tarefas de forma permanente.
- Navegação entre páginas (Tarefas, Concluídas, Lixeira).

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** → Estrutura das páginas.
- **CSS3** → Estilização com um visual moderno e descontraído.
- **JavaScript (ES6)** → Lógica e interatividade.
- **LocalStorage** → Persistência de dados direto no navegador (sem complicação).

---

## 📂 Estrutura do Projeto
- `Página 1.html` → Saudação inicial e coleta do nome.
- `Página 2.html` → Criação de tarefas.
- `Página 3.html` → Lista principal de tarefas (CRUD).
- `Concluídas.html` → Tarefas finalizadas.
- `Lixeira.html` → Tarefas excluídas.
- Arquivos `.css` → Estilos de cada página.
- Arquivos `.js` → Lógica de cada página.

---

## 🎨 Design
- Fundo com imagem personalizada.
- Interface moderna com caixas de diálogo estilizadas.
- Menu lateral para navegação.
- Botões coloridos e com efeitos visuais para cada ação (✔ concluir, ✎ editar, ✖ excluir, ↩ restaurar).
